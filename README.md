# Escape Sequences in C

| Escape Sequence | Character Description        |
|------------------|------------------------------|
| `\b`             | Backspace                    |
| `\f`             | Form feed                    |
| `\n`             | Newline                      |
| `\r`             | Carriage return              |
| `\t`             | Horizontal tab               |
| `\v`             | Vertical tab                 |
| `\\`             | Backslash                    |
| `\'`             | Single quotation mark        |
| `\"`             | Double quotation mark        |
| `\?`             | Question mark                |
| `\0`             | Null character               |
